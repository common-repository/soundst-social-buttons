== Description ==

This plugin will add a Facebook, Twitter and email share buttons after the body content on pages and posts

== Installation ==

1. Upload the soundst-social-buttons folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Adjust the plugin settings
